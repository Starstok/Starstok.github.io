repo sync 源码出现错误部分解决方法

1.repo sync

出现错误

error: .repo/manifests/: contains uncommitted changes

解决方法：

de>cd .repo/manifests

git stash

git clean -f -d

2.repo sync中途出现error: error: Exited sync due to fetch errors

解决方法：

repo sync -jx -f –force-sync x的数值根据你电脑的配置来决定
